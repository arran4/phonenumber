# Phone number 

Nokia keypad simulate / generator for a meme.. 

## Usage:

### Command line

```bash
$ drawphonecli -outfile out.png -text "Hello how are you?" -fontsize 12
'4433555.555666 446669 277733 99966688?'
Wrote: out.png
```
